
# SharePoint Task Management System (Demo)

This is a simple demonstration of a task management system similar to what can be built using Microsoft SharePoint.

## Features
- Task tracking
- Task assignment
- Priority management
- Status tracking
- Simple dashboard view

## Files
index.html – Demo dashboard  
tasks.json – Task data

## How to Run
1. Download the project
2. Open index.html in a browser

## How this connects to SharePoint
In SharePoint the same structure would be implemented using:

• SharePoint Lists for tasks  
• Choice columns for Priority and Status  
• Person column for Assigned To  
• Power Automate for task notifications  
• SharePoint dashboard page for reporting
